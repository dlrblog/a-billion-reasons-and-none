Almost there ✦

Fill in your Firebase config at the top of this file, then follow these steps:

  1. Go to firebase.google.com → create a project named whatever you like
  2. Build → Firestore Database → Create database (choose test mode)
  3. Firestore → Rules → replace with:
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /memories/{doc} {
          allow read, write: if true;
        }
      }
    }
  4. Firestore → Indexes → Add composite index
    Collection: memories  |  Fields: date ASC + createdAt ASC
  5. Authentication → Settings → Authorized domains → add your custom domain
  6. Project Settings → Your apps → Add web app → copy config → paste into FIREBASE_CONFIG at top of this file
  7. Put this file at calendar/index.html in your GitHub repo → push → enable GitHub Pages 🎉
